文献阅读_Macroscopic Description for Networks of Spiking Neurons

发表于2025-10-05|更新于2025-10-14|论文
|浏览量:

🧠 论文基本信息

  • 标题：Macroscopic Description for Networks of Spiking Neurons
  • 作者：Ernest Montbrió1, Diego Pazó2, and Alex Roxin3
  • 发表出处：Physical Review X
  • 发表时间：2015

公式推导

  1. 单个QIF神经元膜电位动力学

$$ \dot { V } _ { j } = V _ { j } ^ { 2 } + I _ { j } , \quad \mathrm { i f } \ V _ { j } \geq V _ { p } , \quad \mathrm { t h e n } \ V _ { j } \leftarrow V _ { r } . $$

其中有如下规则： $V_p = -V_r \to \infty$

  1. 单个神经元的输入电流组成
    $$ I _ { j } = \eta _ { j } + J s ( t ) + I ( t ) , $$

3.平均突触激活的定义

$$ s ( t ) = \frac { 1 } { N } \sum _ { j = 1 } ^ { N } \sum _ { k \backslash t _ { j } ^ { k } < t } \int _ { - \infty } ^ { t } d t ^ { \prime } a _ { \tau } ( t - t ^ { \prime } ) \delta ( t ^ { \prime } - t _ { j } ^ { k } ) . $$

$a _ { \tau }$:突触激活函数

$\delta ( t ^ { \prime } - t _ { j } ^ { k } )$:意味着在$t ^ { \prime } = t _ { j } ^ { k }$处有一个”无限高，无限窄”的尖峰，其总面积为1。这意味着它只在脉冲发生的那一刻才有值，其他时刻都为0.

  1. 连续性方程

$$ \partial _ { t } \rho + \partial _ { V } [ ( V ^ { 2 } + \eta + J s + I ) \rho ] = 0 $$

核心思想：在某个区域内，如果守恒量的密度随时间变化，那么这种变化必然是由于守恒量的流入和流出该区域造成的。

对于一维系统：$\frac { \partial _ { \rho } } { \partial _ { t } } + \frac { \partial _ { J_x } } { \partial _ { x } } = 0 $
其中$\rho$是密度，$J_x$是沿$x$方向的通量（密度乘以速度）

  1. 膜电位V在时间t的条件概率密度函数

$$ \rho ( V | \eta , t ) = \frac { 1 } { \pi } \frac { x ( \eta , t ) } { [ V - y ( \eta , t ) ] ^ { 2 } + x ( \eta , t ) ^ { 2 } } $$

  1. LA的半宽高

$$ x ( \eta , t ) = \pi r ( \eta , t ) . $$

推导过程

$$
\begin{aligned}
r(\eta, t) & = \lim_{V \rightarrow \infty} [\rho(V \mid \eta, t) \dot{V}(V \mid \eta, t)] \
& = \lim_{V \rightarrow \infty} \left[ \left( \frac{1}{\pi} \frac{x(\eta, t)}{[V - y(\eta, t)]^2 + x(\eta, t)^2} \right) \cdot \left( V^2 + I_j \right) \right] \
& = \lim_{V \rightarrow \infty} \left[ \frac{1}{\pi} \frac{x(\eta, t)}{V^2} \cdot V^2 \right] \quad \text{since } V \rightarrow \infty \
& = \frac{1}{\pi} x(\eta, t)
\end{aligned}
$$

  1. 总瞬时发放率

$$ r ( t ) = \frac { 1 } { \pi } \int _ { - \infty } ^ { \infty } x ( \eta , t ) g ( \eta ) d \eta . $$

  1. 平均膜电位

$$ y ( \eta , t ) = \mathrm { p . v . } \int _ { - \infty } ^ { \infty } \rho ( V | \eta , t ) V d V . $$

属于第一类反常积分，可使用柯西积分的方式来求解。

$E_(x)=\int x f_(x) d x . $

  1. 宏观平均膜电位

$$ v ( t ) = \int _ { - \infty } ^ { \infty } y ( \eta , t ) g ( \eta ) d \eta . $$

  1. 引入复数变量$w ( \eta , t )$

$$ \partial _ { t } w ( \eta , t ) = i [ \eta + J s ( t ) - w ( \eta , t ) ^ { 2 } + I ( t ) ] , $$

证明过程

由连续方程4
$$ \partial _ { t } \rho + \partial _ { V } [ ( V ^ { 2 } + \eta + J s + I ) \rho ] = 0 , $$

记：$ (a(t)\equiv \eta+J s(t)+I(t))$
由LA分布5
$$ \rho(V|\eta,t)=\frac{1}{\pi}\frac{x(\eta,t)}{(V-y(\eta,t))^2+x(\eta,t)^2}$$

记：$(D(V,t)\equiv (V-y)^2+x^2)$

$\partial_t\rho$的计算：

$$
\rho=\frac{1}{\pi}\frac{x}{D}\quad\Rightarrow\quad
\partial_t\rho=\frac{1}{\pi}\frac{\partial_t xD - x\partial_t D}{D^2}.
$$
而
$$
\partial_t D = -2(V-y),\partial_t y + 2x,\partial_t x.
$$
于是分子变为
$$
\begin{aligned}
\partial_t xD - x(-2(V-y)\partial_t y + 2x\partial_t x)\
= \partial_t x(D-2x^2) + 2x(V-y)\partial_t y.
\end{aligned}
$$

计算$\partial_V[(V^2+a)\rho$

$\begin{aligned} \partial_V \rho & =\frac{1}{\pi} \partial_V\left(\frac{x}{D}\right) \ & =\frac{1}{\pi} x \cdot(-1) D^{-2} \cdot 2(V-y) \ & =-\frac{2 x(V-y)}{\pi D^2}\end{aligned}$

然后

$
\begin{aligned}
\partial_V[(V^2+a)\rho]& =(V^2+a)\partial_V\rho + 2V\rho\
& = -\frac{2x(V-y)(V^2+a)}{\pi D^2} + \frac{2xV}{\pi D}.
\end{aligned}
$

把两项代回方程为$\partial_t\rho+\partial_V[(V^2+a)\rho]=0$,两边同时乘以 $\pi D^2$：

$$
\partial_t x (D-2x^2)+2x(V-y)\partial_t y-2x(V-y)(V^2+a)+2xV D=0
$$

$$
D=(V-y)^2+x^2=V^2-2Vy+y^2+x^2
$$

$$
\Rightarrow \partial_t x \big[(V^2-2Vy+y^2+x^2)-2x^2\big]+2x(V-y)\partial_t y-2x(V-y)(V^2+a)+2xV(V^2-2Vy+y^2+x^2)=0
$$

$$
\Rightarrow \partial_t x (V^2-2Vy+y^2-x^2)+2x(V-y)\partial_t y-2x(V-y)(V^2+a)+2xV(V^2-2Vy+y^2+x^2)=0
$$

展开所有括号:

$$
\begin{aligned}
&\partial_t x V^2 - 2y \partial_t x V + \partial_t x (y^2-x^2) \
&+ 2xV\partial_t y - 2x y\partial_t y \
&- 2xV^3 + 2x y V^2 - 2x a V + 2x a y \
&+ 2xV^3 - 4x y V^2 + 2x(y^2+x^2)V = 0
\end{aligned}
$$

合并同类项:

  • $V^3: (-2x + 2x)=0$
  • $V^2: \partial_t x + (2x y) - (4x y)=0 \Rightarrow (\partial_t x - 2x y)V^2$
  • $V^1: (-2y \partial_t x) + 2x\partial_t y - 2x a + 2x(y^2+x^2)=0$
  • $V^0: \partial_t x (y^2-x^2) - 2x y\partial_t y + 2x a y=0$

多项式形式:
$$
(\partial_t x - 2xy)V^2 + [-2y \partial_t x + 2x(\partial_t y - a + y^2 + x^2)]V + [\partial_t x (y^2 - x^2) - 2x y\partial_t y + 2x a y]=0
$$

令各系数为零:

  • $V^2$ 系数:
    $$\boxed{\partial_t x = 2xy}$$

  • $V^1$ 系数:
    $$
    \begin{aligned}
    &-2y(2xy) + 2x(\partial_t y - a + y^2 + x^2)=0 \
    \Rightarrow &-4x y^2 + 2x(\partial_t y - a + y^2 + x^2)=0 \
    \Rightarrow &\partial_t y - a + y^2 + x^2 - 2y^2 = 0 \
    \Rightarrow &\boxed{\partial_t y = y^2 - x^2 + a}
    \end{aligned}
    $$

  • $V^0$ 系数:
    $$\partial_t x (y^2-x^2) - 2x y\partial_t y + 2x a y=0$$
    代入前两式后自动成立。

令复变量

$$
w(\eta,t) \equiv x(\eta,t) + i y(\eta,t).
$$

则

$$
\partial_t w = \partial_t x + i \partial_t y = 2xy + i \big( y^2 - x^2 + a \big).
$$

$$
w^2 = (x+iy)^2 = x^2 + 2 i x y - y^2 \quad \Rightarrow \quad -w^2 + a = a - x^2 - 2 i x y + y^2.
$$

两边乘以 $i$：

$$
i \big( a - w^2 \big) = i (a - x^2 + y^2) + 2 x y,
$$

因此

$$
\boxed{\partial_t w(\eta,t) = i \big( \eta + J s(t) - w(\eta,t)^2 + I(t) \big).}
$$
这正是论文中的 Eq.(10)

  1. 最终形式

[
\left{
\begin{aligned}
\dot{r} &= \dfrac{\Delta}{\pi} + 2rv, \[6pt]
\dot{v} &= v^{2} + \bar{\eta} + Jr + I(t) - \pi^{2}r^{2}.
\end{aligned}
\right.
]

证明过程
$$
g(\eta) = \frac{1}{\pi} \frac{\Delta}{(\eta - \bar{\eta})^2 + \Delta^2}
$$

$$
r(t) = \frac{1}{\pi} \int_{-\infty}^{\infty} x(\eta,t) g(\eta) d\eta
$$

$$
v(t) = \int_{-\infty}^{\infty} y(\eta,t) g(\eta) d\eta
$$

$$
\pi r(t) + i v(t) = \int_{-\infty}^{\infty} w(\eta,t) g(\eta) d\eta
$$

$w(\eta,t)$ 解析于下半平面且在 $|\eta| \to \infty$ 时衰减。

$$
g(\eta) = \frac{1}{\pi} \frac{\Delta}{(\eta - \bar{\eta})^2 + \Delta^2} = \frac{1}{2\pi i} \left[ \frac{1}{\eta - (\bar{\eta} - i\Delta)} - \frac{1}{\eta - (\bar{\eta} + i\Delta)} \right]
$$

闭合积分路径于下半平面，仅包围极点 $\eta = \bar{\eta} - i\Delta$。

$$
\oint w(\eta,t) g(\eta) d\eta = 2\pi i \cdot \text{Res}[w(\eta,t) g(\eta), \eta = \bar{\eta} - i\Delta]
$$

$$
\text{Res}[w(\eta,t) g(\eta), \eta = \bar{\eta} - i\Delta] = \frac{1}{2\pi i} w(\bar{\eta} - i\Delta, t)
$$

$$
\therefore \quad \int_{-\infty}^{\infty} w(\eta,t) g(\eta) d\eta = w(\bar{\eta} - i\Delta, t)
$$

$$
\boxed{\pi r(t) + i v(t) = w(\bar{\eta} - i\Delta, t)}
$$

由

$$
\partial_t w(\eta,t) = i \big( \eta + J s(t) - w(\eta,t)^2 + I(t) \big)
$$

$$
\Rightarrow \quad \partial_t w(\bar{\eta} - i\Delta, t) = i \big( \bar{\eta} - i\Delta + J s(t) - w(\bar{\eta} - i\Delta, t)^2 + I(t) \big)
$$

令

$$
w(\bar{\eta} - i\Delta, t) = \pi r(t) + i v(t)
$$

$$
\Rightarrow \quad \partial_t [\pi r(t) + i v(t)] = i \big[ \bar{\eta} - i\Delta + J s(t) - (\pi r(t) + i v(t))^2 + I(t) \big]
$$

$$
\Rightarrow \quad \pi \dot{r} + i \dot{v} = i(\bar{\eta} + J s + I) + \Delta - i(i v + \pi r)^2
$$

分离实部与虚部：

$$
\begin{cases}
\dot{r} = \dfrac{\Delta}{\pi} + 2rv, \[6pt]
\dot{v} = v^2 + \bar{\eta} + J s(t) + I(t) - \pi^2 r^2.
\end{cases}
$$

将$s(t)$换成$r(t)$
$$
\boxed{
\begin{aligned}
\dot{r} &= \frac{\Delta}{\pi} + 2rv, \
\dot{v} &= v^2 + \bar{\eta} + J r + I(t) - \pi^2 r^2,
\end{aligned}
}
$$

公式符号解释

符号 解释
$V_j$ 第 $j$ 个神经元的膜电位
$\dot{V_j}$ 膜电位 $V_j$ 对时间的导数
$I_j$ 第 $j$ 个神经元的输入电流
$V_p$ 神经元膜电位的峰值
$V_r$ 神经元膜电位的重置值
$\eta_j$ 第 $j$ 个神经元外部输入的异质性、淬火部分
$J$ 突触权重
$s(t)$ 平均突触激活，表示网络中所有神经元突触活动的平均水平
$I(t)$ 共同的、随时间变化的外部输入分量
$N$ 神经元总数
$t_k^{(j)}$ 第 $j$ 个神经元的第 $k$ 次脉冲发放时间
$\delta(t)$ 狄拉克 $\delta$ 函数
$a_{\tau}(t)$ 由单个突触前脉冲引起的归一化突触激活函数，时间尺度为 $\tau$
$\tau$ 突触激活函数的时间尺度
$\rho(V|\eta, t)$ 在时间 $t$ 时，膜电位在 $V$ 和 $V+dV$ 之间、参数为 $\eta$ 的神经元所占的比例
$g(\eta)$ 神经元异质性参数 $\eta$ 的概率分布函数
$\partial_t$ 对时间 $t$ 的偏导数
$\partial_V$ 对膜电位 $V$ 的偏导数
$x(\eta, t)$ Lorentzian 分布的半宽度
$y(\eta, t)$ Lorentzian 分布的中心，代表给定 $\eta$ 值下神经元的平均膜电位
$r(\eta, t)$ 给定 $\eta$ 值下的神经元发放率
$r(t)$ 总发放率
$v(t)$ 总平均膜电位
$w(\eta, t)$ 复数形式的宏观变量，定义为 $x(\eta, t) + iy(\eta, t)$
$\tilde{\eta}$ Lorentzian 分布 $g(\eta)$ 的中心
$\Delta$ Lorentzian 分布 $g(\eta)$ 的半宽度
